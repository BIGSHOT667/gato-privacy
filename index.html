Ce que Gato
stocke sur toi

Mis à jour en juin 2025 — Bot Discord de tickets
⚠️
À lire avant tout. Gato est un bot de tickets de support. Contrairement à la plupart des bots, il stocke le contenu textuel de tes messages dans les tickets, ainsi que ton nom d'utilisateur Discord et ton avatar. C'est nécessaire pour générer les transcriptions.

L'essentiel

Gato gère les tickets de support du serveur (demandes staff, abus, signalements). Chaque message envoyé dans un ticket est sauvegardé en base de données pour générer une transcription consultable par le staff.

Données collectées

  • Discord User ID
    Identifiant numérique unique Discord. Utilisé pour identifier l'auteur du ticket et les intervenants.
  • Nom d'utilisateur Discord (tag)
    Ton pseudo Discord actuel au moment de l'envoi du message.
    Stocké dans la transcription
  • URL d'avatar Discord
    Lien vers ta photo de profil Discord, affiché dans les transcriptions HTML.
    Stocké dans la transcription
  • Contenu de tes messages
    Texte intégral de ce que tu écris dans le salon ticket.
    ⚠️ Données sensibles — stockées pour génération de transcription
  • Raison du ticket
    Texte que tu saisis lors de l'ouverture du ticket.
    Stocké pour le suivi du staff
  • Type de ticket
    Catégorie choisie : staff, abus ou signalement.
  • Qui a ouvert / fermé / claim
    User IDs des intervenants et timestamps d'ouverture/fermeture.
Gato ne collecte jamais : email, adresse IP, numéro de téléphone, ou toute donnée en dehors du contexte de ces tickets Discord.

Pourquoi ces données

Sans les messages sauvegardés, il n'est pas possible de générer une transcription du ticket. Le staff du serveur en a besoin pour traiter les demandes, vérifier les abus et protéger les membres.

Qui peut voir tes données

Les transcriptions sont visibles uniquement par les membres du staff du serveur Discord qui disposent des rôles appropriés. Aucun tiers n'y a accès.

Les données ne sont jamais partagées, vendues ni transmises à l'extérieur du serveur d'hébergement du bot.

Durée de conservation

Les données sont conservées tant que le bot est actif sur le serveur. Il n'existe pas de suppression automatique. L'administrateur peut purger manuellement la base de données à tout moment.

Tes droits (RGPD)

Si tu es résident européen, tu disposes d'un droit d'accès, de rectification et d'effacement de tes données. Pour l'exercer, contacte un administrateur du serveur Discord concerné — il peut supprimer toutes les entrées te concernant dans la base de données.

Contact

Pour toute demande relative à tes données, contacte directement l'administrateur de ton serveur Discord ou rejoins le serveur de support du bot.
